## KVM VPS 是什么？为什么比 OpenVZ 更值得选

KVM VPS（Kernel-based Virtual Machine）是一种完全虚拟化技术，每台 VPS 拥有独立的内核空间，支持加载任意操作系统。

简单总结：KVM VPS 的隔离性更强，资源不和邻居共享，可以跑 Docker、WireGuard 等需要内核权限的服务，适合绝大多数实际使用场景。搬瓦工目前全部套餐均基于 KVM 架构，并搭配自研的 KiwiVM 控制面板，支持一键重装系统、机房迁移、快照备份等操作。
